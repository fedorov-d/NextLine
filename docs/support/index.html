Common Issues

  • If nearby stops are not showing, make sure Location Access is enabled for NextLine Transit in iPhone Settings > Privacy & Security > Location Services.
  • If favorites are missing from the app, widget, or Apple Watch, open the iPhone app once and allow it to finish loading and syncing.
  • If Apple Watch data looks outdated, make sure the iPhone app has been opened recently and watch connectivity is available.
  • If arrival times or route data look incorrect, include the stop, route, and approximate time when reporting the issue.

Transit Data

Transit information depends on upstream transit data sources. Delays, schedule changes, or incomplete data may affect what appears in the app.
